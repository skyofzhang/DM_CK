# 面板规格：弹幕滚动条面板

> Canvas: Barrage_Canvas (L_BARRAGE=20) | 基准: 1080×1920
> 引用: `docs/modules/ui_theme.md`

---

## 面板总体布局

| 属性 | 值 |
|------|-----|
| 位置 | Anchor底部拉伸，Y=1460（距底部460px） |
| 尺寸 | 1080×200px |
| 背景 | 无（透明，弹幕浮在游戏画面上） |

---

## 弹幕滚动条

### 显示规格
| 属性 | 值 |
|------|-----|
| 滚动方向 | 从右向左匀速滚动 |
| 滚动速度 | 150px/秒（观众能读完一条，200ms内） |
| 最大同时显示条数 | 5条弹幕（分行，每行36px高，行间距4px） |
| 弹幕生命周期 | 从右侧出现→滚动至左侧消失，约7秒 |

### 弹幕显示格式
```
[头像24×24] [昵称（F_SMALL,20px）] ：[指令内容（F_BODY,28px,C_TITLE）]
```

### 指令颜色高亮
| 指令 | 颜色 |
|------|------|
| 1 (摸鱼) | C_FIRE_WARM `#FF8C3A` |
| 2 (挖煤) | C_COAL_GRAY `#C8C8C8` |
| 3 (挖矿) | C_ICE_LIGHT `#88CCFF` |
| 4 (生火) | C_FIRE_HOT `#FF6820` |
| 6 (打怪) | C_DANGER_RED `#FF2200` |
| 666 | C_GOLD_ALERT `#FFD700` + 加粗 |
| 无效指令 | C_TEXT_DIM `#6080A0`（暗淡，不高亮） |

### 特殊弹幕显示
- **666弹幕**：在普通弹幕列上方固定位置出现大字"⚡ 全局加速激活！" 1.5s，然后消失
- **新用户加入**：在弹幕上方显示欢迎横幅（独立于滚动条，见下方欢迎横幅）

## 底部守护者贡献排行榜（bobao区上方）

| 属性 | 值 |
|------|-----|
| 位置 | Y=1700（距底部220px），宽1040px，高150px |
| 功能 | 显示本场Top5守护者 |

### 每行结构（从左到右）
```
[排名 24px Bold] [头像 32×32] [昵称 F_SMALL 20px] [积分 F_SMALL 20px] [变化箭头 16px]
```
| 排名 | 底色 |
|------|------|
| 第1名 | 金色边框 `#FFD700` |
| 第2名 | 银色 `#C0C0C0` |
| 第3名 | 铜色 `#CD7F32` |
| 4-5名 | 普通白色 |

## bobao滚动播报

| 属性 | 值 |
|------|-----|
| 位置 | X=20, Y=1340（顶部内容区下方），宽540px，高120px |
| 字体 | F_BODY (28px, C_TEXT_WHITE)，描边1px黑色 |
| 滚动方式 | 从右向左，速度100px/秒 |
| 最大同时条数 | 2条（普通+特殊） |

### bobao类型与颜色
| 类型 | 颜色 | 示例 |
|------|------|------|
| 普通采集 | C_TEXT_WHITE | 无（频率太高不播报） |
| 礼物T3+ | C_POSITIVE `#44FF88` | "感谢 [昵称] 的甜甜圈！" |
| 礼物T4 | C_GOLD_ALERT `#FFD700` + ⚡ | "⚡ 感谢 [昵称] 的能量电池！" |
| 礼物T5 | C_GOLD_ALERT Bold + 字号+50% | "💥 超级感谢 [昵称] 的神秘空投！" |
| 失败警报 | C_DANGER_RED `#FF2200` + 闪烁 | "村子断粮了！" |
| 守护者超越 | C_POSITIVE | "[A] 超过了 [B]！" |
| 主播操作 | `#FFFF00` Bold | "⚡ 主播激活紧急加速！" |
| 新用户加入 | C_ICE_WHITE | "[昵称] 加入了村庄！" |
