# 面板规格：礼物通知系统

> Canvas: Gift_Canvas (L_GIFT=100) | 基准: 1080×1920
> 引用: `docs/modules/ui_theme.md`, `docs/modules/animation_spec.md`
> 技术规范: 所有礼物通知面板预创建在 Gift_Canvas 下，通过 SetActive 控制

---

## 面板总体结构

```
Gift_Canvas (Sort Order=100, 穿透射线)
├── T1_StarParticle     (ParticleSystem预制，位置动态)
├── T2_BorderEffect     (Panel, 预创建，SetActive)
├── T3_GiftBounce       (Panel, 预创建，SetActive)
├── T4_FullscreenGlow   (Panel, 预创建，SetActive)
├── T5_EpicAirdrop      (Panel, 预创建，SetActive)  ← 最顶层
└── GiftBanner_Queue    (滚动横幅队列，最多3条同时显示)
```

## GiftBanner_Queue：礼物横幅队列

| 属性 | 值 |
|------|-----|
| 位置 | 屏幕左侧中部，X=20, Y=-700, 宽540px |
| 最大同时显示 | 3条（新条插入队列顶部，旧条向下移） |
| 每条高度 | 80px |
| 每条背景 | 渐变：左侧礼物等级色(80%) → 右侧透明 |

**每条横幅结构**（从左到右）：
- 礼物等级色彩条（宽8px，满高）
- 礼物图标（40×40px）
- 昵称（F_BODY 28px, C_TEXT_WHITE）+" 送出 " + 礼物名（C_TITLE）
- 最右侧礼物等级标签（"T1"~"T5"，对应颜色小标签）

**进出动画**：
- 进入：从左侧滑入 (X:-540→0, 0.2s, EaseOutCubic)
- 退出：向上淡出 (Alpha 1→0, 0.3s)，在队列满时触发最旧条退出
