# 极地生存法则 — 动画规格 v1.0

> 参考：`docs/modules/ui_theme.md` 中的 A_ 动画变量
> 技术栈：Unity 2022.3 + URP；Worker动画用 Animator状态机 + 代码驱动；UI动画用 DOTween/LeanTween

---

## 1. Worker 动画系统

### 1.1 Worker状态机（5状态）

```
[Idle] → (收到指令) → [Move] → (到达工位) → [Work]
                                              ↓ (CD结束)
                                           [Idle]
[任何状态] → (666弹幕) → [Special: 3秒金光] → (恢复原状态)
[任何状态] → (魔法镜) → [Frozen: 30秒冰冻] → (恢复原状态)
```

### 1.2 每种状态动画规格

#### Idle（待机）
| 属性 | 规格 |
|------|------|
| 动画类型 | 循环 |
| 动画描述 | 原地轻微上下浮动（Bob动画） |
| 参数 | Y偏移: 0→+0.1单位→0, 周期1.0秒, SineCurve |
| 实现方式 | `transform.localPosition += Vector3.up * Mathf.Sin(Time.time * Mathf.PI * 2) * 0.05f` |
| 触发头顶气泡 | 隐藏（SetActive=false） |

#### Move（移动）
| 属性 | 规格 |
|------|------|
| 动画类型 | 单次（到达后自动转Work） |
| 动画描述 | 向目标工位移动，直线路径 |
| 移动速度 | 3单位/秒 |
| 实现方式 | `Vector3.MoveTowards(transform.position, targetPos, speed * Time.deltaTime)` |
| 朝向 | `transform.LookAt(targetPos)`（仅Y轴旋转） |
| 完成判定 | `Vector3.Distance < 0.1f` |
| 头顶气泡 | 显示目标工作类型emoji |

#### Work（工作）
| 属性 | 规格 |
|------|------|
| 动画类型 | 循环（持续至CD结束） |
| 动画描述 | 工具挥动：Capsule绕Z轴左右摆动 |
| 摆动角度 | Z轴 -30° → +30° |
| 摆动速度 | 0.5秒/次（完整摆动1秒）|
| 实现方式 | `transform.localRotation = Quaternion.Euler(0, 0, Mathf.Sin(Time.time * Mathf.PI * 2) * 30)` |
| 每种工作变体 | 见下方工作变体表 |

##### 工作变体（头顶气泡颜色）
| 指令 | 工位 | 气泡颜色 | 气泡图标 |
|------|------|---------|---------|
| 1 采食物 | 鱼塘 | `#44AAFF` | 🐟 |
| 2 挖煤 | 煤矿 | `#666666` | ⛏ |
| 3 挖矿 | 矿山 | `#88CCFF` | 🪨 |
| 4 生火 | 炉灶 | `#FF6820` | 🔥 |
| 6 打怪 | 城门 | `#FF2200` | ⚔️ |

#### Special（特殊：666/主播加速）
| 属性 | 规格 |
|------|------|
| 动画类型 | 单次3秒后恢复原状态 |
| 动画描述 | Worker自身材质发出金色光晕 |
| 实现方式 | 切换材质为 GlowMaterial（Emission金色），3秒后恢复原材质 |
| 光晕颜色 | `#FFD700`，强度HDR×3 |
| 粒子 | 可选：2-3个金色粒子从Worker底部向上漂浮，3秒消散 |
| 全体同步 | 所有活跃Worker同时触发，时间轴一致 |

#### Frozen（冻结：魔法镜）
| 属性 | 规格 |
|------|------|
| 动画类型 | 单次30秒后恢复 |
| 动画描述 | Worker原地被冰晶覆盖，停止所有动作 |
| 实现方式 | 停止所有状态机更新；切换为FrozenMaterial（蓝色半透明）；ParticleSystem播放冰晶特效 |
| 视觉 | 材质颜色 `#88CCFF`，透明度60% |
| 粒子 | 持续蓝色冰晶粒子围绕，速度慢（speed=0.2） |
| 头顶气泡 | 显示❄️图标，冷蓝色背景 |

## 2. UI动画规格

### 2.1 资源数值变化动画

| 触发 | 动画 | 参数 |
|------|------|------|
| 资源增加 | 数字跳动 | 当前值→新值，0.3s，EaseOutQuart |
| 资源增加大量（礼物）| 数字跳动+绿色闪光 | 0.5s，跳动完成后+色 |
| 资源减少 | 数字变红 + 跳动 | `C_NEGATIVE`色，0.3s |
| 资源低于阈值 | 红色闪烁 | A_FLASH_WARN，0.5s周期，循环至恢复 |
| 资源为0 | 持续红色+字体加粗 | 静止，直到恢复 |

### 2.2 炉温数值变化

| 炉温范围 | 颜色 | 特效 |
|---------|------|------|
| > 0℃ | 橙色 `#FF8C3A` | 无 |
| -50℃~0℃ | 蓝色 `#88CCFF` | 无 |
| -80℃~-50℃ | 冷蓝闪烁 `#4A90D9` | A_FLASH_WARN 1s周期 |
| < -80℃ | 冷紫闪烁 `#7766DD` | A_FLASH_WARN 0.5s周期 + sfx_cold_alarm |

### 2.3 守护者排名变化

| 触发 | 动画 | 参数 |
|------|------|------|
| 积分增加（无排名变化） | 积分数字跳动 | A_NUM_ROLL 0.3s |
| 排名上升 | 行向上平移 | A_RANK_CHANGE 0.5s，新位置绿色↑箭头1s |
| 排名下降 | 行向下平移 | A_RANK_CHANGE 0.5s，新位置红色↓箭头1s |
| 进入Top3 | 行缩放+颜色高亮 | Scale 1→1.1→1, 0.5s，名字变金色 |
| T5礼物送出 | 该行金色边框闪烁 | A_FLASH_WARN 0.3s周期，持续10s |

### 2.4 昼夜切换动画

| 阶段 | 动画 | 时长 |
|------|------|------|
| 白天→夜晚 | 屏幕整体蓝色overlay淡入（Alpha 0→0.2）；背景BGM淡入淡出 | 3s过渡 |
| 夜晚→白天 | 蓝色overlay淡出（Alpha 0.2→0）；BGM淡入淡出 | 2s过渡 |
| 夜晚开始 | 全屏闪光（白色→暗，0.5s）+ "第X夜" 大字淡入淡出（3s） | — |
| 白天开始 | 全屏金色光芒扫过（0.5s） + "第X天" 大字淡入淡出（3s） | — |

## 3. 礼物特效动画规格（详细时间轴）

### T1 仙女棒（0.5秒）
```
0.0s  ——  ParticleSystem启动（5颗星形粒子，从头像位置出现）
0.0s  ——  粒子以30°/帧旋转，向外扩散半径50px
0.5s  ——  粒子Alpha渐出0→消失
```

### T2 能力药丸/魔法镜（2秒）
```
0.0s  ——  4个角落ParticleSystem启动（每角15粒子，向中心运动）
0.5s  ——  粒子到达中心，中心光环Image(Size 0→300px, 0.3s, EaseOutBack)
0.8s  ——  光环保持，昵称字条从左边滑入（A_SLIDE_UP 0.2s）
1.5s  ——  光环开始淡出（Alpha 1→0, 0.5s）
2.0s  ——  消失
[魔法镜额外]：中心冰晶Particle爆发，被冻结玩家名字蓝色❄️闪烁2s
```

### T3 甜甜圈/超能喷射（3秒）
```
0.0s  ——  礼物大图标(240×240)从屏幕左外侧飞入中央（EaseOutBounce, 0.5s）
0.5s  ——  图标到达中央，抖动动画（Scale 1→1.15→1, 0.3s, ×3次）
0.5s  ——  bobao播报出现（A_SLIDE_UP）
1.4s  ——  爆炸粒子从图标中心爆发（20颗金色粒子，向四周散射）
1.4s  ——  图标Scale 1→2.0 + Alpha 1→0（0.5s, EaseOutQuart）
2.0s  ——  粒子继续飞散（0.5s）
2.5s  ——  所有特效消失
3.0s  ——  完成，恢复游戏正常渲染
```

### T4 能量电池（5秒）
```
0.0s  ——  全屏橙色遮罩淡入（Alpha 0→0.4, 0.3s, EaseInQuart）
0.0s  ——  屏幕整体后处理色温+30（0.3s过渡）
0.3s  ——  能量电池图标(320×320)Scale 0→1.2→1（EaseOutBack, 0.3s）
0.3s  ——  进度条Slider从0→100%（3s线性）
0.5s  ——  bobao播报出现（⚡黄色粗体）
3.5s  ——  进度条满格，数字闪烁3次，炉温图标跳动A_WAVE_BOUNCE
4.5s  ——  图标Alpha 1→0（0.5s）
4.5s  ——  全屏遮罩Alpha 0.4→0（0.5s）
4.5s  ——  色温恢复（0.5s）
5.0s  ——  完成
```

### T5 神秘空投（8秒）★
```
0.0s  ——  服务器发送GIFT_PAUSE=3000ms（游戏逻辑暂停）
0.0s  ——  黑色遮罩淡入（Alpha 0→0.85, 0.3s, EaseInQuart）
0.3s  ——  飞机SFX + 空投箱从屏幕顶部Y+2000开始下落
0.5s  ——  空投箱下落（EaseInCubic, 1s, 到达屏幕中央Y=960）
1.5s  ——  着地震动（屏幕震动：X±5px, 0.2s）
1.5s  ——  爆炸ParticleSystem：金色烟花，100颗粒子，扩散半径500px
1.5s  ——  资源图标从箱子飞出散落（食物/煤/矿/盾牌图标各1个）
2.0s  ——  "[昵称] 拯救了村庄！"大字出现（Scale 0→1, EaseOutBack, 0.5s）
2.0s  ——  字体：F_GIFT_T5_NAME（80px, C_GOLD_ALERT）
2.0s  ——  守护者排名：该玩家行金色边框开始闪烁
2.5s  ——  bobao超级播报（3s持续）
3.0s  ——  GIFT_PAUSE结束，游戏逻辑恢复（视觉特效继续）
5.0s  ——  大字Alpha 1→0（1s）
6.0s  ——  烟花粒子消散（0.5s）
7.0s  ——  遮罩Alpha 0.85→0（0.5s）
8.0s  ——  完全恢复游戏正常渲染
注意：视觉特效持续8s，但游戏逻辑在3s时就恢复
```

## 4. 主播按钮动画

### ⚡ 紧急加速按钮
| 状态 | 动画 |
|------|------|
| 可用 | 脉冲效果：Scale 1→1.05→1, 1s周期，C_GOLD_ALERT光晕 |
| 点击 | A_BTN_PRESS（Scale 1→0.9, 0.1s）|
| 激活后3s | 全体Worker金色光晕（Special动画） |
| CD中 | 灰色遮罩覆盖，倒计时数字从120倒数 |
| CD完成 | 灰色遮罩消失 + 脉冲效果恢复 |

### 🌊 触发事件按钮
| 状态 | 动画 |
|------|------|
| 可用 | 轻微波纹效果（ripple圆圈扩散，2s周期） |
| 点击 | A_BTN_PRESS |
| CD中 | 灰色遮罩 + 倒计时60s |
