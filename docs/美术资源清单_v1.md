# 《极地生存法则》美术资源清单 v1.0

> 生成日期：2026-04-23
> 对标策划案：`极地生存法则_策划案.md` v1.27
> 适用：美术 / 音频 / 特效团队按此清单交付；开发团队按此命名匹配代码路径
> 统一字体：`Assets/Resources/Fonts/AlibabaPuHuiTi-3-85-Bold.otf`（已集成，TMP SDF 已生成）
> 所有 PNG 默认 sRGB / Alpha=Transparency；所有音频默认 44.1kHz

## 命名与路径规范

- **PNG 图标 / Sprite**：
  - 小图标（≤128px）：`icon_<模块>_<名>.png`
  - UI 背景：`bg_<模块>_<名>.png`
  - 按钮：`btn_<名>_<状态>.png`（状态：normal/pressed/disabled）
- **3D 模型 Prefab**：`<模块>_<名>_<等级>.prefab`（如 Gate_Lv1.prefab）
- **特效 Prefab**：`VFX_<模块>_<名>.prefab`（如 VFX_FrostPulse.prefab）
- **音频**：`sfx_<模块>_<名>.mp3` / `bgm_<场景>.mp3`
- **字体资产**：`<字体名> SDF.asset`

## 规格缩写

- 三角面：Tris
- 骨骼数：Bones
- 贴图尺寸：Tex (如 Tex 1024)
- 采样率：44.1k
- 时长：单次 / Loop

## 八、美术与开发对接要点

1. **所有 TMP 文本使用 `AlibabaPuHuiTi-3-85-Bold SDF.asset`**，美术不要自行引入其他中文字体
2. **PBR 贴图 4 套**（Albedo / Normal / Metallic / Roughness），Unity URP 标准通道
3. **透明 PNG 要求 Alpha=Transparency**（不是 Alpha Is Transparency 导入选项，而是源文件本身）
4. **特效时长**：UI 弹出 0.3~0.5s；伤害飘字 1.2s；升级特效 1~2s；建造进度持续 180~300s
5. **Prefab 命名 大小写敏感**：`Gate_Lv1.prefab` 而不是 `gate_lv1.prefab`
6. **场景挂载**：由开发团队按 Editor 脚本 `Tools → DrscfZ → Run All Section Setups` 自动完成；美术不直接编辑 MainScene.unity
7. **版本追踪**：本清单为 v1.0，后续策划案更新会同步调整；美术如发现命名冲突或规格不合理，及时反馈给开发团队
