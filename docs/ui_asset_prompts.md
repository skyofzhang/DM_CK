# 极地生存法则 · UI资源生成提示词文档
> 工具：Lovart | 分辨率基准：1080×1920px 竖屏 | 风格：冰雪写实游戏UI
> 更新：2026-03-10

---

## 一、使用说明

### 工作流建议
1. **先生成效果图** → 确认整体视觉方向后
2. **再按效果图拆散图** → 保证散图风格与效果图一致
3. 每次输入提示词时，先粘贴 **§全局基础参数**，再粘贴对应的具体提示词

### 文件命名与存储结构
```
Assets/Art/UI/
├── screens/          效果图（全屏合图）
├── panels/           面板底图
├── hud/              战斗HUD组件
├── icons/            图标
├── buttons/          按钮
├── bars/             进度条 / 滑块
├── notifications/    通知横幅
├── ranking/          排行相关
├── decorations/      装饰元素
└── backgrounds/      背景 / 遮罩
```

## 二、§全局基础参数（每次生成前必须先告知Lovart）

```
【项目基础风格】
游戏名称：极地生存法则
平台：抖音直播互动游戏
美术风格：冰雪写实游戏UI，介于写实与卡通之间，有质感但不失清晰易读性
主题意象：极地严寒、中世纪冰雪要塞、生存挑战、凛冬守城
整体气质：硬派、寒冷、史诗感，带有轻微厚重的磨砂质感

【色彩系统】
主背景色：深海蓝 #0D1F33
面板底色：深蓝灰 #132840（通常带60~80%不透明度）
主强调色：冰晶蓝 #4A9ECC
高光色：冰雪白 #E8F4FF
边框色：霜蓝 #7EC8E3
副文字色：浅冰蓝 #93C5FD
资源-食物：暖琥珀 #F5A623
资源-煤炭：炭灰 #6B7280
资源-矿石：金属银 #94A3B8
资源-温度（热量）：危险红 #EF4444
城门血量：生命绿 #22C55E
成功/胜利：金色 #F59E0B
危险/失败：深红 #DC2626
文字主色：纯白 #FFFFFF

【字体规范】
标题：粗体，带轻微冰刻压印感，字间距略宽
正文/数值：清晰无衬线粗体，游戏数字感
不使用过度花哨的艺术字，优先保证可读性

【通用设计语言】
面板：磨砂冰晶质感，四角带冰棱装饰，内侧轻微冷色内发光
按钮：石刻冰雕风格，有立体浮雕感，上下边缘有高光
边框：霜晶纹理，细线描边，局部有冰晶结构细节
装饰：雪花、冰晶、霜纹、极地要塞符文
禁止使用：高饱和暖色大面积填充、卡通描边风、扁平纯色块

【基准画布】
竖屏 1080×1920px，72dpi
散图要求透明背景（PNG格式），效果图为不透明完整页面
```

## 六、Lovart 使用技巧提示

1. **每次对话开头粘贴** §全局基础参数，确保风格统一
2. **一次只生成一个元素**，避免复杂度过高导致质量下降
3. **效果图生成后**，用"参考这张效果图，提取其中的XXX部分，单独生成透明背景散图"可保证风格一致
4. **如果生成结果偏暖或偏卡通**，追加关键词：`不要卡通风格，不要暖色，深蓝冷色系，写实游戏UI，磨砂质感`
5. **尺寸不符时**，在提示词末尾加：`请严格按照Xpx×Xpx输出，不要改变比例`
