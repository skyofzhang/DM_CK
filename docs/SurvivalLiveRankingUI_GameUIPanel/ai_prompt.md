# SurvivalLiveRankingUI_GameUIPanel AI 效果图提示词

## 使用方式 / 注意事项

- 使用方式：把同目录下的 `ui_layout_prototype_SurvivalLiveRankingUI_GameUIPanel.png` 作为参考图上传，再配合下面的主提示词使用。
- 目标：生成一张可以指导 Unity prefab 美术落地的 1080x1920 竖屏 UI 效果图。
- 注意：参考图里的虚线、坐标、英文 prefab 名、黑色施工标题条都是施工标注，最终效果图不要照搬成灰盒线框。
