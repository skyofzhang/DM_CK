## 命名规范（硬约束，写错脚本找不到）

- **Prefab**：`<模块>_<子类>_<名>.prefab`（`Gate_Lv5.prefab` / `Building_Watchtower.prefab`）
- **图标 Sprite**：`<Category>_<Name>.png`（`Gift_Icon_1.png` / `icon_shop_A1_worker_pep_talk.png`）
- **UI 背景**：`bg_<模块>_<名>.png`
- **特效 Prefab**：`VFX_<模块>_<名>.prefab`（`VFX_FrostPulse.prefab`）
- **动画 FBX**：`<Name>_<Action>.fbx`（`kuanggong_01_Attack.fbx`）
- **音效 SFX**：`sfx_<system>_<action>.mp3`（`sfx_gift_t6_legendary.mp3`）
- **BGM**：`bgm_<phase>.mp3`（`bgm_season1_day.mp3`）
- **UI 音效**：`ui_<action>.ogg/.mp3`
- **字体资产**：`<字体名> SDF.asset`
- **光照 Profile**：`Lighting_<主题>_<Phase>.asset`
- **URP Volume Profile（后处理）**：`PPVol_<主题>.asset`
- **Prefab 大小写敏感**：`Gate_Lv1.prefab` ≠ `gate_lv1.prefab`

## 规格缩写

- `Tris` 三角面 | `Bones` 骨骼数 | `Tex 1024` 主贴图尺寸 | `44.1k` 采样率
- 图标默认 `128×128px PNG`；UI 背景 `1920×1080` 或百分比锚定；建筑图标 `256×256`

## 状态标签

- ✅ **已就位**：磁盘实际存在且被代码引用，无需重做
- ⚠️ **部分占位**：Editor 脚本已创建占位节点（纯色块 / 默认字体），美术替换 Sprite 即可
- ❌ **缺失**：需美术制作交付
- 🗑️ **弃用**：角力游戏时代遗留或被替代，不要再制作

## 美术与开发对接要点

1. **字体**：所有 TMP 文本使用 `AlibabaPuHuiTi-3-85-Bold SDF.asset`，不自行引入其他中文字体（2026-04-23 全量扫描 1490 font + 629 outline material 已统一）
2. **PBR 贴图**：Albedo / Normal / Metallic / Roughness，Unity URP 标准通道
3. **透明 PNG**：要求源文件本身带 Alpha=Transparency（非 Unity 导入选项）
4. **特效时长基准**：UI 弹出 0.3-0.5s；伤害飘字 1.2s；升级 VFX 1-2s；建造 180-300s；Loop 特效明确标注
5. **Prefab 大小写敏感**：`Gate_Lv1.prefab` ≠ `gate_lv1.prefab`，严格按本清单
6. **场景挂载**：由开发团队用 Editor 脚本 `Tools → DrscfZ → Run All Section Setups` 自动完成；美术不直接编辑 `MainScene.unity`
7. **`douyin_id` 全部 TBD**：上线前必须抖音平台回填真实 ID；测试期服务器按 `price_fen` 兜底匹配
8. **Prefab Variant 优先**：§30 皮肤、§31 怪物变种优先用 Prefab Variant 而非新建 Prefab，减少重复资源开销
9. **LOD Group 保留**：所有角色 Prefab 的 LOD 设置不要删除，性能优化依赖（屏占比 <5% 自动裁剪）
10. **阴影投射一律关闭**：所有 kuanggong 系列 SMR `shadowCastingMode = Off`（性能优化 2.9 FPS → 20+ FPS 的关键之一）
