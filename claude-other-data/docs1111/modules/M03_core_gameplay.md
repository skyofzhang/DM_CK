# M03: 核心玩法

## 赚钱逻辑
核心战斗要有"爽感"——秒怪→推图→卡关→变强→继续推。卡关就是付费点。

## 依赖模块
M01（视觉吸引）, M02（互动入口）

## 战斗系统

### 自动战斗
- 加入后角色自动攻击+释放技能，无需玩家手动操作
- 普攻: 持续自动，单目标
- 技能: CD=7.8~8.8秒，倍率1.8~2.2x，打4个目标
- 元素技能: 冰/毒/雷/火四种，由时装(Avatar)决定
- 战斗完全服务端计算，客户端只展示

### 伤害公式
```
基础伤害 = ATK × 技能倍率
暴击伤害 = 基础伤害 × 1.5(暴击倍率)
最终伤害 = 暴击伤害 × (1 - 目标免伤率) × (1 + 伤害加成%)
```

## 关卡体系

### 主线关卡
- 总计500+关，分5个难度大区
- 每区100关，难度系数递增

| 难度 | 关卡范围 | 难度系数 | 怪物特征 |
|------|---------|---------|---------|
| 普通 | 1-100 | 1.0~2.59 | 基础怪物 |
| 困难 | 101-200 | 2.85~7.40 | 暴击率提升 |
| 精英 | 201-300 | 8.14~21.14 | 增加闪避 |
| 史诗 | 301-400 | 23.25~60.40 | 增加免伤 |
| 传奇 | 401-500 | 66.44~115.19 | 全属性强化 |

### 每关结构
- 3波制: 第1波小怪(3-4只) → 第2波小怪(5-6只) → 第3波Boss
- 每波有战斗时间限制
- 怪物从怪物池(10种)中随机抽取

### 怪物分类 (78种)
| 类型 | ID范围 | 数量 | 特征 |
|------|--------|------|------|
| 小怪 | 40001~40040 | 40 | HP=1000, ATK=50, 暴击5%~10% |
| 关卡Boss | 41001~42025 | ~50 | HP=1000~1亿, 按难度递增 |
| 通天塔Boss | 42101~42105 | 5 | HP=1000万~9.99亿 |
| 世界Boss | 42201~42206 | 6 | HP=1亿~20亿, 免伤72% |

### Boss机制
- 多段血条: 每段100万HP(BossHpPerTube=1000000)
- 隐藏Boss: 关卡有概率出现隐藏Boss，30秒选择时间(HideBossChooseTime=30)
- 隐藏Boss奖励更丰厚（额外经验+稀有掉落）

## 复活系统
- 角色死亡后等待3秒(RevivalTimeGap=3)自动复活
- 普通复活: 免费，回满血
- 高级复活: 消耗高级复活道具(20026)，复活带buff
- PVP复活限制: 最多3次(PVPRevLimit=3)
- 礼物复活: 1币仙女棒=复活×1

## 奖励系统
- 基础奖励: 每关固定(BaseRewardItemId) → 经验(20011) + 金币(20012) + 材料
- 排名加成: BaseRewardRankBonus → [1st=1.5x, 2nd=1.3x, 3rd=1.2x, 4th=1.1x, 5th=1.0x]
- 首次通关额外奖励
- 装备掉落: 关卡越高品质越好
