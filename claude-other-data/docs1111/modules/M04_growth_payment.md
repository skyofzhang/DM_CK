# M04: 成长付费

## 赚钱逻辑
14条养成线=14个付费坑。每条线都能通过礼物/充值加速。玩家永远有"再强一点就能过关"的驱动力。

## 依赖模块
M03（核心玩法）

## 1. 角色等级系统

### 境界体系 (20境界)
练气→筑基→金丹→元婴→化神→炼虚→合体→大乘→散仙→地仙→天仙→金仙→太乙→大罗→准圣→圣人→天道→鸿蒙→无极→归零

- 每境界约25级（共500+级）
- 每境界突破需要对应品阶的突破石:
  - 初级突破石(20001): 练气~金丹
  - 中级突破石(20002): 元婴~合体
  - 顶级突破石(20003): 大乘以上

### 属性成长 (每级)
- Lv1基础: HP=1600, ATK=240, DEF=88
- 每级增长: HP+160, ATK+20, DEF+7
- 经验需求: Lv1=100, 每级约增长20%（O(n²)曲线）
- 暴击/闪避: 固定初始值，靠装备和宝石提升

### 9个装备部位

### 强化 (40星)
- 每星+5%属性 → 40星=+200%
- 消耗: 强化石(20005~20007) + 金币
- 概率: StarLvUpProb = [失败%, 保持%, 成功%]
- 失败不掉星（只浪费材料）

### 附魔 (20级)
- 每级+20%属性(血/攻/防三选一)
- 100%成功率
- 消耗: 附魔石(20004) + 金币

### 吞噬
- 用低品质装备喂给高品质装备
- 提升固定属性（不受概率影响）

## 战力计算
战力 = 角色等级属性 + 装备总属性×(1+星级%+附魔%) + 宝石加成 + 圣翼加成 + 坐骑加成 + 宠物属性 + 时装属性 + 称号加成 + 道纹加成 + 光环加成 + 套装效果
