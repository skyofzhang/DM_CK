### 2. 通用组件规范

#### 关闭按钮 (CloseButton)
- 类型: Button + Image, 锚点(1,1)→(1,1), pivot(1,1)
- 位置: (-16,-16) 相对父面板右上角, 尺寸64×64
- 图标: "✕"矢量图白色, 背景圆形#000000 80%
- 按下缩放: 0.95, 0.1s

#### 标题栏 (TitleBar)
- 锚点(0,1)→(1,1), 高度88px
- 背景: 横向渐变 #1A0A2E → #0D1A3A → #1A0A2E, Alpha=220
- 底边分隔线: 1px #6644AA
- 标题文字: 32px Bold #FFD700 居中

#### 品质边框
- Image Sliced九宫, 边框3px, 圆角6px

| 品质 | 边框颜色 | 背景底色(5%透明) |
|------|----------|------------------|
| 白(普通) | #FFFFFF | #FFFFFF0D |
| 绿(精良) | #00FF00 | #00FF000D |
| 蓝(优秀) | #4488FF | #4488FF0D |
| 紫(史诗) | #CC44FF | #CC44FF0D |
| 橙(传说) | #FF8800 | #FF88000D |
| 红(神器) | #FF0000 | #FF00000D |
| 金(古神) | #FFD700 | #FFD7000D |
| 暗金(限定) | 渐变#FFD700→#FF8800 | — |

#### 红点提示 (RedDot)
- 无数字:20×20 / 有数字:32×20胶囊形
- 锚点(1,1), 偏移(+8,+8)相对按钮右上角
- 背景#FF3333, 文字16px White Bold, 最大"99+"

### 4. 动画规范

| 动画类型 | 参数 | Easing |
|---|---|---|
| 面板进入 | X+540→0, 0.3s | EaseOutCubic |
| 面板关闭 | X=0→+540, 0.2s | EaseInCubic |
| 弹窗出现 | Scale 0→1, 0.2s | EaseOutBack(1.2) |
| 弹窗关闭 | Scale 1→0, 0.15s | EaseInBack |
| Toast淡入 | Alpha 0→1, 0.2s | Linear |
| Toast停留 | 1.5s | — |
| Toast淡出 | Alpha 1→0, 0.3s | Linear |
| 按钮按下 | Scale 1→0.95, 0.1s | Linear |
| 数字跳动 | 数值插值, 0.5s | EaseOutQuart |
