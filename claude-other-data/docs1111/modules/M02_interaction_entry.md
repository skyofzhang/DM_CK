# M02: 互动入口

## 赚钱逻辑
门槛越低，参与人数越多，潜在付费用户越多。一条弹幕就能加入。

## 依赖模块
M08（技术底座）

## 弹幕指令（5条）

| 指令 | 功能 | 场景 | 说明 |
|------|------|------|------|
| 加入 | 创建角色/重新加入 | 任意 | 首次自动创建角色,分配初始装备 |
| 查角色 | 查看角色信息 | 任意 | 返回: 境界/战力/装备/宝石概览 |
| 查背包 | 查看背包 | 任意 | 返回: 道具列表+数量 |
| 查战宠 | 查看宠物 | 任意 | 返回: 宠物名/等级/属性 |
| 开箱子 | 开箱操作 | 任意 | 消耗箱子道具,获得随机物品 |

## 加入流程
1. 观众在直播间发送弹幕"加入"
2. 服务器收到弹幕 → 检查房间人数(≤20)
3. 若首次: 创建角色(RoleId=1001) + 分配白装一套 + 初始宠物青鸾
4. 若已有角色: 加载存档数据
5. 客户端: Spine角色出现在战场，加入自动战斗
6. HUD显示: 玩家头像+名字+境界名+战力值

## 人数限制
- 每直播间最多20名玩家同时参与战斗
- 观众上限30人(ViewerLimit=30)
- 超出人数时弹幕"加入"返回提示"当前房间已满，请等待空位"
- 玩家5分钟无操作自动踢出（释放位置）

## 主播管理
- 主播可通过后台踢出指定玩家
- 被踢玩家数据保留，可重新加入
- 主播可设置房间为仅关注可加入

## 新手引导
- 首次加入: 自动展示3秒弹窗说明核心指令
- 赠送新手礼包: 50000金币 + 100突破石 + 初级强化石×50
- 自动穿戴白色品质装备（9部位全套）

- [ ] 弹幕"加入"→角色出现在战场
- [ ] 房间满20人时拒绝新加入+提示
- [ ] 首次加入获得新手礼包
- [ ] 弹幕"查角色"返回正确信息
- [ ] 5分钟AFK自动踢出
