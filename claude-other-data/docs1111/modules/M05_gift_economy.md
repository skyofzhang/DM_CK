# M05: 礼物经济

## 赚钱逻辑
礼物是唯一的收入来源。每档礼物必须有即时可见的"超值回馈感"，让观众觉得花钱=爽。

### 场景差异机制
同一份礼物在不同游戏场景(Scene)触发不同效果:
- Scene 1: 主线战斗 → 伤害/技能类效果
- Scene 2: 大厅/待机 → Buff/复活类效果
- Scene 3: PVP → 复活/强化类效果
- 每个CGiftId在GiftEffectTable中有多条记录，按Scenes字段区分

### Buff效果
- GiftBuff: 属性提升比例(0~1%)
- GiftTime: 持续时间(60秒)
- GiftMax: 叠加上限(200)

## 商城系统

- 货币: 商城积分(道具20013), 由礼物产出
- 每日兑换上限, 防止囤积
- 价格参考:
  - 突破石: 10~20积分
  - 强化石: 10~20积分
  - 附魔石: 800积分(稀缺)
  - 宠物碎片: 50积分

## 夺宝系统

- 触发: 99币礼物=1次抽奖
- 36种奖品, 概率权重(万分制)
- 高概率(60%+): 碎片、突破石、强化石
- 中概率(20%): 10级宝石、中级材料
- 低概率(5%): 仙灵、限定时装
- 保底: 万法归宗称号 = 累计16700次

### 幸运值
- 悟性丹(19币礼物)累计提升幸运值
- 幸运值提升稀有物品掉率上限0~30%

## 经济闭环
```
礼物(人民币) → 商城积分 + 碎片 + 复活 + VIP经验
    ↓
商城积分 → 突破石/强化石/附魔石/碎片
碎片 → 圣翼/坐骑/宠物进阶
VIP → 每日奖励 + 专属特权
    ↓
全部 → 战力提升 → 推更高关卡 → 更好掉落 → 继续需要更多提升
```
